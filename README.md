# Truffler Server

Truffler Server Laravel

## Installation

Install [Composer](https://getcomposer.org/Composer-Setup.exe)

## Clone Project

```bash
git clone https://github.com/HelloVVolf/truffler-api
```

## Command
Install composer
```bash
composer install
```

Copy .env
```bash
cp .env.example .env
```

Make database
```bash
php artisan key:generate
```

Make database
```bash
php artisan migrate
php artisan db:seed
```

Run server
```bash
php artisan serve
```
